===== 짧은 설명 (최대 132자) =====

유튜브 영상의 속도, 음계(피치), 구간 반복을 자유롭게 조절하세요. 강화된 루프 보호로 다음 곡 넘어가기 걱정 없이 완벽한 구간 연습이 가능합니다.


===== 상세 설명 =====

🎵 Video Controller - 유튜브 고급 비디오 컨트롤러

음악 연습, 어학 학습, 강의 시청을 위한 최고의 유튜브 컨트롤러입니다!


✨ 주요 기능

🎛️ 전역 ON/OFF 마스터 스위치
• 모든 기능을 한 번에 켜고 끌 수 있습니다
• 문제 발생 시 즉시 원래 상태로 복원
• ON/OFF 상태가 자동으로 기억됩니다


🚀 정밀한 재생 속도 조절
• 0.1배속 단위로 세밀하게 조절 (0.25x ~ 4.0x)
• 50%, 60%, 70%, 80%, 90%, 100%, 110%, 125%, 150%, 175%, 200% 프리셋
• +/- 버튼으로 빠른 조절
• 속도 기능만 끄고 다른 기능은 사용 가능


🎵 음계(피치) 조절
• 반음 단위로 음계 변경 (-24 ~ +24 반음, 2옥타브)
• 음악 연습에 최적: 노래 키를 내 음역대에 맞춰 조절
• 악기 연습: 원곡의 키를 악기에 맞춰 변경
• 현재 음계명 표시 (C, C#, D, D#, E, F, F#, G, G#, A, A#, B)
• AudioWorkletNode 최신 기술 사용으로 성능 향상


🔄 완벽한 구간 반복
• 다음 곡으로 절대 넘어가지 않음 (강화된 보호 시스템)
• 자동재생 목록에서도 한 곡만 반복
• 슬라이더로 구간 설정 또는 시간 직접 입력 ("1:30" 또는 "90")
• 현재 재생 위치 표시 (노란색 인디케이터)
• 시작점/끝점으로 즉시 이동


🔄 스마트 곡 변경 감지 & 자동 리셋
• 5가지 방법으로 곡 변경 감지 (제목, URL, 시간 점프, 끝→처음, 종료 이벤트)
• 곡이 바뀌면 자동으로 기본값으로 리셋
• 깨끗한 상태로 새 곡 시작
• 2초 간격 빠른 감지


💡 사용 사례

🎸 음악 연습
• 어려운 구간만 반복 재생하여 집중 연습
• 키를 낮춰서 편안하게 따라 부르기
• 속도를 줄여서 빠른 연주 구간 천천히 익히기

📚 어학 학습
• 리스닝 구간 반복으로 정확한 발음 학습
• 속도 조절로 초보자도 쉽게 따라가기
• 중요한 표현만 반복 학습

🎓 강의/교육
• 중요한 개념 구간 반복 청취
• 빠른 강의는 천천히, 느린 강의는 빠르게
• 완벽한 이해를 위한 구간 학습


🛡️ v4.2.0 버그 픽스 및 개선사항

✅ 루프 끝점 드래그 시 다음 곡 넘어가는 버그 완전 해결
✅ 슬라이더 UI가 화면을 벗어나는 문제 수정
✅ 루프 보호 강화 (0.3초 → 0.5초 선제 차단, 50ms → 30ms 인터벌)
✅ AudioWorkletNode로 마이그레이션 (Deprecated 경고 제거)
✅ 곡 변경 감지 개선 (5초 → 2초 간격)
✅ 곡별 설정 저장 제거 (항상 기본값으로 시작)
✅ 전역 ON/OFF 토글 UI 개선
✅ 코드 정리 및 최적화 (3KB 감소)


🎯 완벽한 호환성

• YouTube 완벽 지원
• Chrome Extension Manifest V3
• React 18 기반 현대적인 UI
• Web Audio API 최신 기술


📖 사용 방법

1. YouTube 영상 재생
2. 브라우저 우측 상단의 익스텐션 아이콘 클릭
3. 팝업에서 원하는 기능 사용
   - 속도: +/- 버튼 또는 프리셋 클릭
   - 피치: 반음/한음/옥타브 버튼으로 조절
   - 구간 반복: 슬라이더나 시간 입력으로 구간 설정 → 반복 시작


⚠️ 주의사항

• YouTube에서만 작동합니다
• 일부 DRM 보호 콘텐츠는 지원되지 않을 수 있습니다
• 피치 조절 시 원본 비디오는 음소거되고 변환된 오디오가 재생됩니다
